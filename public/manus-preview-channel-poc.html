Manus preview channel PoC

This page hosts the remote script used to test whether code running in a Manus web preview frame can send trusted SpacePreviewerChannel messages to the desktop client.

Direct browser visits are expected to be harmless. The PoC is meaningful only when the script executes in the outer document of a Manus web preview iframe.
